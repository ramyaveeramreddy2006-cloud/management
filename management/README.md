# TaskForge Management Website

TaskForge is a static task-management website built with HTML, CSS, and JavaScript.

## File Structure

```text
management
├── index.html
├── style.css
├── script.js
├── images
│   └── logo.png
└── README.md
```

## Features

- User authentication section concept
- Create, complete, filter, and delete tasks
- Progress counters for total, open, and done tasks
- Responsive design for desktop and mobile screens
- Clean structure for GitHub upload

## Run Locally

Open `index.html` directly in your browser.

You can also use VS Code Live Server if installed.
